# 📄 API Curriculum Maker

Backend da aplicação para gerar currículos personalizados em formato PDF, desenvolvido em **Java** e **Spring Boot**.

## 📝 Descrição

A **API Curriculum Maker** expõe uma API REST que recebe dados do usuário e gera um currículo minimalista em PDF utilizando a biblioteca **iText PDF**.

## ✨ Funcionalidades

- 🛠️ Recebe dados do usuário via API REST.
- 📄 Gera automaticamente um arquivo PDF de currículo a partir dos dados fornecidos.
- 🔍 Permite visualização dos dados do currículo.

## 🛠️ Tecnologias Utilizadas

- **Java**
- **Spring Boot**
- **iText PDF** (para geração do PDF)
